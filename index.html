eerie Vibes: Interactive Art Generator

eerie Vibes began as an experiment in prompt engineering and AI collaboration. Using GPT-4, I created an interactive art generator without writing any code directlyâ€”instead, relying entirely on prompt engineering to develop a program that generates eerie, algorithmic art.

Over two hours of focused prompt engineering and troubleshooting, I guided an AI to create a program that combines Python, PIL, and Pygame to generate atmospheric, sinister artwork. The program features 10 distinct drawing functions, each creating unique spooky images like sinister cats and eerie suns, all mapped to user input through a custom GUI interface.

Key Components

  • GUI Setup: Pygame-based interface with text input and submission button
  • Image Generation: 10 custom draw_image functions creating unique spooky imagery
  • Dynamic Backgrounds: Randomly generated dark, muted shapes for atmosphere
  • Interactive Elements: Random eerie prompts and real-time image generation

Technical Implementation

Python Pygame PIL Vector Graphics Algorithmic Art Prompt Engineering GPT-4

Development Process

This project demonstrates the potential of prompt engineering in software development. Through careful instruction and iteration with GPT-4, I was able to create a fully functional art generator that captures a specific aesthetic without writing any code directly. The process revealed both the capabilities and limitations of current LLMs in creative software development.
